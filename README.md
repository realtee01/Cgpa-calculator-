# T/A | Professional CGPA Calculator 🎓

A sleek, school-portal-inspired CGPA calculator built with **HTML5**, **Tailwind CSS**, and **Vanilla JavaScript**. Designed for students who want a fast, responsive, and personalized way to track their academic performance.

![Vercel Deployment](https://img.shields.io/badge/Vercel-000000?style=for-the-badge&logo=vercel&logoColor=white)
![Tailwind CSS](https://img.shields.io/badge/Tailwind_CSS-38B2AC?style=for-the-badge&logo=tailwind-css&logoColor=white)

---

## ✨ Key Features

* **Personalized Entry:** Custom "T/A" branded login screen that greets you by name.
* **Dynamic Course Management:** Add or remove course rows instantly with a clean table interface.
* **Dual Theme Support:** High-contrast **White UI** and a premium **Dark Mode** for late-night study sessions.
* **Real-time Logic:** Automated calculations based on the standard 5.0 CGPA scale.
* **Fully Responsive:** Optimized for mobile, tablet, and desktop views.
* **Data Persistence:** Uses `localStorage` to remember your name and theme preference even after refreshing.
